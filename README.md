# AIPharmAccount - Pharmacy Management System

Open-source** pharmacy invoice, inventory, and business intelligence system.

## Features
- 📦 Inventory Management
- 💰 Sales & Purchase Billing
- 📊 AI-Powered Analytics (Claude API)
- 📧 Email Invoices (Gmail OAuth)
- 📈 Sales Reports & Predictions
- 🔐 User Management

## Tech Stack
- Java 21
- Spring Boot 3.x
- JavaFX (UI)
- Sqlite
- Maven
- Flyway (Migrations)
